__Preliminary Documentation__ - 2019-11-23
____  
__ldc-browser: The LDC (Linux Docker Container) Web Browser project.__  

Current Development Version: __0.1.0-b1__  
Latest Release Version: __n/a__  

____  
____  
The __[LDC Browser](https://github.com/ewsdocker/ldc-browser/wiki")__  project currently produces _Docker_ containers for 4 well-known browsers:

<ul>
  <dl>  
    <dt><a href="https://github.com/ewsdocker/ldc-browser/wiki/FirefoxQuantum">Firefox Quantum</a></dt>  
      <dd>"The latest Firefox brings privacy protections front and center, 
          letting you track the trackers."
      </dd>  
    <br>  
    <dt><a href="https://github.com/ewsdocker/ldc-browser/wiki/NetSurf">Net Surf</a></dt>  
      <dd>"NetSurf is a free, open source web browser. 
          It is written in C and released under the GNU Public Licence, 
          version 2. NetSurf has its own layout and rendering engine and is
          small and capable of handling many web standards."
      </dd>  
    <br>  
    <dt><a href="https://github.com/ewsdocker/ldc-browser/wiki/PaleMoon">Pale Moon</a></dt>  
      <dd>"Pale Moon is an Open Source, Goanna-based web browser 
          built from its own, independently developed source ... and 
          focuses on efficiency and ease of use."
      </dd>  
    <br>  
    <dt><a href="https://github.com/ewsdocker/ldc-browser/wiki/Waterfox">Waterfox Classic</a></dt>  
      <dd>"Waterfox does not collect tracking or usage information, and
          supports 64-Bit NPAPI plugins (such as FlashPlayer).  Explore 
          the web the way it was meant to be!"
      </dd>  
  </dl>  
</ul>  

The __ldc-browser__ project is a member of the __LDC Stack__ project.  The goal of the __LDC Stack__ project is to make it easier to release new _Docker_-based containers by creating specialized __LDC Stack__ containers (or _Stack Blocks_) designed for this purpose.

__Getting Released Versions__  
The current source shown on this page is the __Current Development Version__, and is available through the green "Clone or download" button on this page.   All __Release Version__s are shown by selecting the _releases_ item above the green bar at the top of this page.  

Pre-built _Docker_ containers are available from [ewsdocker/ldc-browser](https://hub.docker.com/r/ewsdocker/ldc-browser) on [Docker Hub](https://hub.docker.com).  If a __Latest Release Version__ is not found on the _Docker Hub_ site, then it has not yet been released.
